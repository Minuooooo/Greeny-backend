## 컨벤션
### Issues
- 기능 개발, 오류 수정 등의 이슈 생성
- 이슈 번호 확인
- 트러블 슈팅 기록

### Branch
-Type/#Issue.number-Title
- master: 최종본
- env: 환경 설정
- feat: 이슈 별 기능 개발
- refactor: 리팩토링
- fix: 오류 수정
- test: 테스트 코드 작성

### Commit
-[Type] Title
- env: 설정
- feat: 기능 개발
- refactor: 리팩토링
- fix: 오류 수정
- chore: gradle 설정, 기타 작업
- test: 테스트 코드

### PR
-[#Issue.number] Type: Title
